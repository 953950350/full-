# baixiu
### 说明：这里是百秀的完整项目，静态页面的展示： https://953950350.github.io/baixiu/

百秀项目是一个完整的后台管理系统，能够实现对整个网站的管理。

本项目所采用技术有：

  - 静态页面使用boostrap搭建
  - 后端采用php技术
  - 文章页面分页功能是使用php代码实现
  - 评论页面的分页功能是使用ajax技术和web本地储存和twbs-pagination分页插件实现
  - 图片上传部分使用ajax异步上传图片
  - 首页使用echarts插件实现数据的可视化（这个在完整的项目里面实现的，静态页面中没有）
  - 本项目前端后端所有的逻辑都是由本人实现
